## 🖥️ What Local-Chat Does

Local-Chat is a simple chat app that runs on your own computer. It uses Streamlit to show a chat page in your browser, so you can open it like a normal website on your PC.

It is built for local use on Windows. That means the chat stays on your machine and does not need a public server. It is a good fit if you want a private chat space, a test chat page, or a local web app you can open fast.
