# Локальный чат /b/ (MichiTheCat)

Простой локальный чат на базе **Streamlit**, который работает в вашей локальной сети.  
Идеально подходит для общения без доступа к глобальному интернету – например, в поездках или при отключении внешней сети.

## Как это работает

Вы запускаете сервер на ноутбуке, подключаете телефон через USB (режим модема или USB-тетеринг) и создаёте локальную сеть.  
На телефоне (или другом устройстве) открываете браузер, переходите по адресу ноутбука в локальной сети – и можете общаться с кем угодно в этой сети.

## Возможности

- Отправка сообщений с указанием ника (или анонимно)
- Просмотр истории сообщений в виде таблицы
- Автоматическое сохранение истории в файл `History.json`
- Обновление ленты сообщений одной кнопкой
- Работает на любом устройстве с браузером в одной сети

## Технологии

- [Streamlit](https://streamlit.io/) – быстрый интерфейс на Python
- JSON – хранение истории сообщений

## Установка и запуск

1. Убедитесь, что у вас установлен Python 3.7 или выше.
2. Склонируйте репозиторий или скачайте файлы проекта.
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
   или вручную:
   ```bash
   pip install streamlit
   ```
4. Запустите сервер:
   - **Windows**: дважды щёлкните `start.bat`
   - **Linux/macOS**: выполните в терминале `bash start.sh`
   
   Или напрямую:
   ```bash
   streamlit run app.py --server.address 0.0.0.0 --server.port 8501
   ```

5. Откройте браузер и перейдите по адресу:
   ```
   http://<IP-адрес-ноутбука>:8501
   ```
   (например, `http://192.168.42.1:8501`, если ноутбук раздаёт сеть через USB)

## Структура проекта

- `app.py` – основной код чата
- `History.json` – файл с историей сообщений (создаётся автоматически)
- `requirements.txt` – список зависимостей
- `start.bat` / `start.sh` – скрипты для быстрого запуска
- `ReadMe.md` – этот файл

## Пример использования

1. Запускаете сервер.
2. На ноутбуке вводите своё прозвище и сообщение, нажимаете «Отправить».
3. Друзья на других устройствах открывают ту же страницу и видят сообщения сразу после обновления (кнопка «Обновить сообщения» или автоматическое обновление через перезагрузку страницы).

## Примечания

- Для работы в локальной сети телефон должен быть подключён к ноутбуку через USB с включённым режимом модема (USB-тетеринг). Либо можно создать Wi-Fi точку доступа на телефоне и подключить ноутбук к ней.
- Порт по умолчанию – `8501`. При необходимости его можно изменить в скриптах запуска.
- История хранится в формате JSON, её легко можно очистить, удалив файл `History.json`.
